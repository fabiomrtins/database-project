### Exercícios Realizados

1. **Liste os produtos e o nome de seus fornecedores.**

   - Arquivo: `raffael-marinho/01.sql`
   - Descrição: A query lista os produtos e o nome dos fornecedores da tabela `suppliers`.

2. **Recupere os pedidos com os nomes dos clientes e dos funcionários responsáveis.**

   - Arquivo: `raffael-marinho/02.sql`
   - Descrição: A query recupera os pedidos juntamente com os nomes dos clientes e dos funcionários responsáveis.
   - Dificuldade: Tive dificuldade em entender a relação entre as tabelas `orders`, `customers` e `employees`, especialmente para associar corretamente os pedidos aos funcionários responsáveis.

3. **Liste os produtos, suas categorias e o nome dos fornecedores.**

   - Arquivo: `raffael-marinho/03.sql`
   - Descrição: A query lista os produtos, suas categorias e o nome dos fornecedores.

4. **Quantos pedidos cada cliente já fez?**

   - Arquivo: `raffael-marinho/04.sql`
   - Descrição: A query retorna o número de pedidos feitos por cada cliente.

5. **Valor total vendido por cada funcionário (considerando os pedidos já enviados).**

   - Arquivo: `raffael-marinho/05.sql`
   - Descrição: A query calcula o valor total vendido por cada funcionário, considerando pedidos enviados.
   - Dificuldade: Tive dificuldade em entender como calcular corretamente o valor total de vendas de um funcionário, especialmente considerando o envio dos pedidos.

6. **Média de preços dos produtos por categoria.**

   - Arquivo: `raffael-marinho/06.sql`
   - Descrição: A query calcula a média dos preços dos produtos por categoria.

7. **Quais os clientes que nunca fizeram pedidos?**
   - Arquivo: `raffael-marinho/07.sql`
   - Descrição: A query retorna os clientes que nunca realizaram um pedido.
